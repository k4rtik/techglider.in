Recommended software for a new Ubuntu install

Friday, April 30, 2010

Here’s my list of recommended software to install after doing a fresh Ubuntu installation:

  • netspeed
  • ntfs-config
  • gparted
  • gnome-do
  • wine
  • vlc
  • gimp
  • filezilla
  • stardict
  • bluefish
  • ccsm
  • opera
  • google chrome
  • chm viewer or chmsee
  • inkscape
  • sunjava
  • acroread
  • python3
  • screem
  • virtualbox
